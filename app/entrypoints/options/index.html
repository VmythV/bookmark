Smart Bookmark — Settings

Privacy

  • Local embeddings: folder names, bookmark titles, and page text are turned into vectors on your device — they are never sent anywhere for the embedding step.
  • What leaves your device: only the LLM re-rank / naming calls, which send the page title/URL and the names of the Top-K candidate folders to the OpenAI-compatible endpoint you configure below. With no API key, the extension uses a local keyword rule and sends nothing.
  • Credentials: API keys and WebDAV/S3 secrets are stored locally in plaintext and are never uploaded by the extension.
